# Agentics Mastery Mission 🐱

> A 3-month deep dive into AI agents, frameworks, leaders, and the future of autonomous systems.
> Designed for ~100 people working through it together.

---

## Mission Overview

**Goal:** Build deep, practical understanding of AI agents and agentic systems
**Duration:** 12 weeks (3 months)
**Daily commitment:** 2-4 hours/day
**Team size:** ~100 people

## Week-by-Week Roadmap

### PHASE 1: Foundation (Weeks 1-3)

#### Week 1: AI/ML Fundamentals
- [ ] Understand transformer architectures
- [ ] Learn about LLMs (GPT, Claude, Gemini, Llama)
- [ ] Study prompt engineering
- [ ] Explore tokenization and embeddings
- [ ] Understand fine-tuning vs RAG vs from-scratch

**Repos to study:**
- `EleutherAI/gpt-neox`
- `huggingface/transformers`

**Reflections to write:**
- What is attention mechanism and why does it matter?
- How do LLMs actually "understand" context?

#### Week 2: Introduction to AI Agents
- [ ] What is an AI agent? (Definition and theory)
- [ ] Agent components: Perception → Planning → Action → Learning
- [ ] Explore existing agent projects
- [ ] Understand tool use and function calling
- [ ] Study agent memory types (short-term, long-term, semantic)

**Repos to study:**
- `significantgravity/promptron`
- `minimax/autonomous-agent`

**Reflections to write:**
- What differentiates an agent from a regular LLM?
- How does memory affect agent capability?

#### Week 3: Agent Frameworks Landscape
- [ ] LangChain fundamentals
- [ ] AutoGPT architecture study
- [ ] BabyAGI core concepts
- [ ] CrewAI multi-agent framework
- [ ] Microsoft Semantic Kernel

**Repos to study:**
- `hwchase17/langchain`
- `Significant-Gravitas/auto-gpt`
- `joaomdmoura/crewai`
- `microsoft/semantic-kernel`

**Reflections to write:**
- Compare 3 agent frameworks - strengths and weaknesses of each
- When would you use one framework over another?
