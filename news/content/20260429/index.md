### GSAR: Typed Grounding for Hallucination Detection and Recovery in Multi-Agent LLMs
**Sources:** Arxiv cs.AI
**Links:** https://arxiv.org/abs/2604.23366
**TL;DR:** A new grounding-evaluation and replanning framework called GSAR is introduced for autonomous multi-agent LLM systems. It evaluates whether claims are grounded in evidence rather than hallucinated model inferences, offering a robust recovery mechanism.
**Agentic Focus:** Multi-agent systems hallucination detection and recovery framework.

### SoccerRef-Agents: Multi-Agent System for Automated Soccer Refereeing
**Sources:** Arxiv cs.AI
**Links:** https://arxiv.org/abs/2604.23392
**TL;DR:** Researchers propose SoccerRef-Agents, a holistic and explainable multi-agent decision-making framework for soccer refereeing. It uses a novel architecture that collaborates via cross-modal RAG to connect visual content with regulatory texts.
**Agentic Focus:** Multi-agent framework combining visual perception and regulatory knowledge reasoning.

### ArguAgent: AI-Supported Real-Time Grouping for Productive Argumentation in STEM Classrooms
**Sources:** Arxiv cs.AI
**Links:** https://arxiv.org/abs/2604.23449
**TL;DR:** ArguAgent is an AI-powered system designed to foster productive STEM argumentation by strategically forming student groups in real-time. It analyzes student arguments and clusters positions to optimize group discussions.
**Agentic Focus:** Generative AI agent applied to educational settings for real-time grouping and argumentation analysis.

### LEGO: An LLM Skill-Based Front-End Design Generation Platform
**Sources:** Arxiv cs.AI
**Links:** https://arxiv.org/abs/2604.23355
**TL;DR:** LEGO is a unified platform for front-end design generation that decomposes the digital workflow into independent steps and represents agent capabilities as standardized composable skills. It features Agent Skill RAG for fast retrieval and significantly improves design automation.
**Agentic Focus:** Composable agent skills and Agent Skill RAG for electronic design automation.
